## Название

АПГ — Альянс партнёров города

## Краткое описание

События, партнёры, эксперты, бонусы и полезные возможности Зеленограда в одном приложении.

## Полное описание

АПГ объединяет жителей, партнёров и экспертов города. В приложении можно
узнавать о событиях и новостях, находить партнёров и специалистов, сохранять
интересные материалы, пользоваться предложениями, накапливать ключи и
общаться с помощником Локи.

Android-версия использует существующий защищённый аккаунт АПГ и синхронизирует
данные с веб-версией.

## Категория

Образ жизни / Городские сервисы.

## Возраст

12+ — предварительно; подтвердить анкетой RuStore.

## Разрешения Android

- Камера — сканирование QR-кодов только после действия пользователя.
- Биометрия — необязательная локальная защита приложения.
- Интернет — загрузка данных АПГ по HTTPS.

Приложение не получает и не хранит биометрические шаблоны. QR-кадр не
сохраняется и не отправляется на сервер.

## Data Safety — черновик

- Передача данных шифруется HTTPS.
- Удаление профиля доступно из приложения.
- Камера используется для QR, без сохранения изображения.
- Биометрические данные обрабатываются только операционной системой.
- Рекламные идентификаторы и Firebase Analytics не используются.
- Push/FCM в версии 1.2.0 не подключены.

## Инструкция модератору

Приложение поддерживает Android 10+. Для базового просмотра доступен гостевой
режим. Для проверки сохранения профиля, выхода и восстановления сессии
предоставляется отдельный тестовый аккаунт без production-персональных данных.
